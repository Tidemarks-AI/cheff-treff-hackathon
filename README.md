# Hackathon Starter

## Welcome

Welcome to the Hackathon Starter project! This is a full-stack template designed to help you get up and running quickly. It includes a React + TypeScript frontend and a backend service, all wired together in a single monorepo.

## Project Structure

```
.
├── frontend/   # React + TypeScript + Vite + shadcn/ui
└── backend/    # Backend service
```

## Getting Started

Install dependencies for both frontend and backend:

```bash
pnpm run install:all
```

Start both services in development mode:

```bash
pnpm run dev
```

Or run them individually:

```bash
pnpm run dev:frontend
pnpm run dev:backend
```
